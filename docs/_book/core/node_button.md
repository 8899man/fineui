# bi.NodeButton

## 表示一个可以展开的节点,用于树状结构的节点元素， 不仅有选中状态而且有展开状态, [BI.BasicButton](/core/basic_button.md)


## API
##### 基础属性
| 参数    | 说明           | 类型  | 可选值 | 默认值
| :------ |:-------------  | :-----| :----|:----
| open | 节点是否展开 | boolean | true,false | false |


## 对外方法
| 名称     | 说明                           |  回调参数     
| :------ |:-------------                  | :-----   
| isOpened | 判断节点是否展开| —
| setOpened | 设置节点展开状态| b
| triggerCollapse | 触发节点收起 | —
| triggerExpand | 触发节点展开| —
